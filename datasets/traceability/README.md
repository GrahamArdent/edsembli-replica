# Traceability Matrix

This folder contains traceability datasets that connect:
- **Frames** → **Indicators** → **Evidence Patterns** → **Comment Templates** → **References**

## Purpose
The traceability matrix enables:
1. Coverage analysis (do all indicators have templates?)
2. Audit trails (which reference supports this template?)
3. Gap identification (which scenarios need more patterns?)

## Files

| File | Format | Description |
|------|--------|-------------|
| `matrix.csv` | CSV | Human-readable traceability export |
| `matrix.parquet` | Parquet | Machine-optimized format for analytics |

## Generation

The matrix is generated by `scripts/generate_matrix.py` (planned).

```bash
python scripts/generate_matrix.py
```

## Schema

| Column | Type | Description |
|--------|------|-------------|
| `frame_id` | string | e.g., `frame.belonging` |
| `indicator_id` | string | e.g., `indicator.belonging.relationships` |
| `evidence_pattern_id` | string | e.g., `evidence.pattern.block_play` |
| `template_id` | string | e.g., `template.comment.belonging.key_learning.01` |
| `ref_ids` | array | Supporting references |
| `section` | string | `key_learning`, `growth`, or `next_steps` |
