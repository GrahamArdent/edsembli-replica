# Tauri + React + Typescript

This template should help get you started developing with Tauri, React and Typescript in Vite.

## Development

From the repo root:

- Preferred (auto-cleans port 1420): `pwsh -File scripts/tauri_dev.ps1`
- Direct: `cd vgreport; npm run tauri -- dev`

If you see `Error: Port 1420 is already in use`, stop the existing Vite dev server (or any process on 1420) and re-run.
