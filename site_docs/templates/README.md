# Templates Library

This folder contains canonical, non-PII template libraries used to produce CoL narrative comments.

## Principles

- No student data or identifying details.
- All templates use `{slot}` placeholders.
- All templates reference at least one indicator and at least one bibliography reference ID (`ref.*`).
